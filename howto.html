遊び方

目的

敵を倒しながらダンジョンを進み、最深部に待つボスを倒すとエンディングになります。
できるだけ多くの GOLD を回収してエンディングを迎えましょう。

ダンジョン

ダンジョンは遊ぶ度に形が変わるランダム生成です。
深い階ほど広くなります。
ゴールに入ると次の階へ進みます。
ゴールがどこにあるか分からないときは、マップ画面 (セレクトボタン) を表示して、まだ端の壁が表示されていない方向へ進みましょう。

戦闘

ダンジョンには敵が多数登場します。
旋回して照準を敵に向け、弾丸を当てて倒しましょう。
自機が敵に接触するとHPが減ります。
HPが0になるとゲームオーバーです。
敵を倒すと GOLD が増えます。

アイテム

敵を倒すとアイテムを落とします。
アイテムを回収すると、自機がパワーアップしたり、GOLDが増えたりします。

ステータス＆セッティング

スタートボタンを押すと、ステータス＆セッティング画面が表示されます。
現在のステータスを確認するとともに、旋回速度や走行速度のセッティングを行うことができます。

» ステータス＆セッティング画面参照。

操作方法

基本操作

1Pコントローラーのみ使用します。



CONTROL TYPE

ステータス＆セッティング画面の "CONTROL TYPE" を変更することで、下記の操作方法に変更できます。

» ステータス＆セッティング画面参照。


画面説明

タイトル画面



画面項目

  • HI-GOLD：ゲーム中で獲得した GOLD の最高記録です。
  • HI-FLOOR：ゲーム中で到達した最深階層です。
  • NEW GAME：パワーアップをリセットしてゲームをスタートします。
  • CONTINUE：パワーアップを維持してゲームをスタートします。

画面特有の操作

  • スタートボタン：ゲームスタート
  • セレクトボタン：NEW GAME または CONTINUE 選択

ダンジョン画面



画面項目

  • HP：自機の耐久力です。表示は「現在HP/最大HP」です。現在HPが0になるとゲームオーバーです。
  • FUEL：自機の燃料です。表示は「現在FUEL/最大FUEL」です。現在FUELが0になるとHPが減っていきます。
  • B 1F：現在の階層です。
  • GOLD：獲得した GOLD です。ボス戦の際はボスの耐久力ゲージに切り替わります。

画面特有の操作

前述の操作方法の通りです。

» 操作方法参照。

ステータス＆セッティング画面



画面項目

  • HP：自機の耐久力です。表示は「現在HP/最大HP」です。
  • FUEL：自機の燃料です。表示は「現在FUEL/最大FUEL」です。
  • ARMS-LV：自機の攻撃力です。
  • GOLD：獲得した GOLD です。
  • FLOOR：現在の階層です。
  • HI-GOLD：ゲーム中で獲得した GOLD の最高記録です。
  • HI-FLOOR：ゲーム中で到達した最深階層です。
  • CONTROL：現在の CONTROL TYPE です。左右キーでA～Fに切り替わります。
  • STEERING：自機の旋回速度です。表示は「現在値/最大値」です。左右キーで最大値以下の値に変更できます。
  • SPEED：自機の走行速度です。表示は「現在値/最大値」です。左右キーで最大値以下の値に変更できます。
  • ITEM-LOCK：ON にすると画面内のアイテムを全て回収するまで隣の画面へ移動しなくなります。左右キーでON/OFFが切り替わります。

画面特有の操作

  • スタートボタン：ダンジョン画面へ戻る
  • 十字キー上下：カーソル移動
  • 十字キー左右：カーソルが当たっている設定値の変更

マップ画面



画面項目

  • B 1F MAP：現在の階層です。
  • マップ上の薄黄色部分：通路です。
  • マップ上の茶色部分：壁です。
  • マップ上の青丸：現在地点です。
  • マップ上の緑四角：ゴール地点です。
  • START BUTTON: TURN OFF BGM：スタートボタンを押す度にBGMを停止/再生します。停止したままダンジョン画面に戻ると効果音のみでゲームを進められます。

画面特有の操作

  • セレクトボタン：ダンジョン画面へ戻る
  • スタートボタン：BGMを停止/再生する
